### üé¨ Movie Search App

A beautiful, modern movie search application built with Node.js and Express that uses the OMDB API to search for movies.

#### Features

- **Beautiful Dark Theme** - Premium glassmorphism design with animated backgrounds
- **Instant Search** - Search millions of movies from the OMDB database
- **Responsive Design** - Works perfectly on desktop and mobile
- **Error Handling** - Gracefully handles searches with no results

#### Tech Stack

- **Backend**: Node.js, Express.js
- **Templating**: EJS
- **API**: OMDB API
- **Styling**: Custom CSS with glassmorphism effects

#### Getting Started

```bash
cd movie_search_app
npm install
node app.js
```

Then open http://localhost:3000 in your browser.
